交互元素表：
| 元素 | 类型 | 说明 |
|------|------|------|
| 大字体开关 | Switch | 即时生效，调 PUT /user/profile {font_size} |
| 通知开关 x3 | Switch | 调 PUT /notification/settings |
| 离线数据管理 | ListItem | 点击→离线数据管理页 |
| 清理缓存 | ListItem | 点击→显示当前缓存大小→确认清理 |

| 退出登录 | Button | 确认弹窗→调 POST /auth/logout→清 token→跳登录页 |

大字体模式：设置 `font_size=large` 时，全局字号 ×1.5，布局自适应

#### 收藏页

要素：
- Tab 切换（食谱/知识）
- 列表展示（图片+标题+摘要）
- 左滑或点击取消收藏

#### 反馈页

要素：
- 反馈历史列表（时间+方案名+评分+AI解读摘要，点击→详情）
- 未反馈列表（当前方案中未反馈的餐，点击→提交反馈）
- 提交反馈：方案/餐选择 + 评分(1-5星) + 症状多选 + 文本描述 + 提交按钮
- 提交后展示 AI 反馈解读

交互规则：
| 操作 | 行为 |
|------|------|
| 选择评分 | 1-5星点击选择 |
| 选择症状 | 多选列表（口干、失眠、乏力等） |
| 提交反馈 | 调 POST /feedback/submit → 展示 AI 解读 |
| AI解读展示 | 在反馈详情中以卡片展示 AI 生成的分析文本 |

#### 食材百科页

要素：
- 列表页：分类筛选（按性味/季节/体质）+ 搜索框 + 结果列表
- 详情页：食材名 + 性味归经 + 功效 + 适用体质 + 禁忌 + 营养 + 时令 + 古籍出处

导航：百科列表 → [点击食材] → 百科详情

#### 养生课堂页

要素：
- 列表页：按主题分类Tab（体质/季节/食材/功效）+ 卡片列表
- 详情页：全屏卡片，标题 + 内容 + 配图，左右滑动翻阅

导航：课堂列表 → [点击卡片] → 全屏详情（左右滑动切换）

#### 全局搜索页

要素：
- 顶部搜索框（自动聚焦 + 防抖 300ms）
- 搜索类型Tab（全部/食谱/食材/知识）
- 搜索历史列表（最近 10 条，可清除）
- 热门搜索词展示（初始状态）
- 搜索结果列表（按类型分组，每组标题+横向3项 或 纵向列表）
- 无结果空状态
- 点击结果跳转对应详情页

交互元素表：
| 元素 | 类型 | 说明 |
|------|------|------|
| 搜索框 | TextInput | 自动聚焦，placeholder "搜索食谱/食材/知识..." |
| 类型Tab | SegmentedControl | 全部/食谱/食材/知识，切换重新搜索 |
| 搜索历史 | List | 最近搜索词，点击重新搜索，右滑删除 |
| 清空历史 | Button | 清空全部搜索历史 |
| 热门搜索词 | Chip Group | 初始展示 TOP 10 热门词，点击搜索 |
| 结果分组 | SectionList | 按类型分组，每组有"查看更多"链接 |
| 结果卡片 | Card | 图片+标题+摘要，点击跳转 |
| 空状态 | View | 搜索无结果时展示空状态插画 |

交互规则：
| 操作 | 行为 |
|------|------|
| 输入关键词 | 防抖 300ms 后调 GET /search/global?keyword=XX |
| 切换类型Tab | 立即按选中类型重新搜索 |
| 点击历史词 | 填入搜索框并触发搜索 |
| 点击结果 | 食谱→食谱详情，食材→百科详情，知识→课堂详情 |
| 清空历史 | 清空本地存储的搜索历史 |

导航：首页→[🔍]→全局搜索页→[点击结果]→对应详情页

#### 消息通知页

要素：
- Tab（全部/未读）
- 通知列表（图标+标题+内容摘要+时间+未读标记）
- 点击→标记已读→跳转相关页面
- 无通知时显示空状态

#### 个人信息编辑页

要素：
- 头像（点击更换）
- 昵称输入框
- 性别选择
- 生日选择器
- 地域选择
- 饮食禁忌设置（过敏/宗教/孕期/其他多选或输入）
- 保存按钮

表单校验：
| 字段 | 规则 |
|------|------|
| 昵称 | 必填，2-20字符 |
