```

│                            │

```

交互元素表：
| 元素 | 类型 | 说明 |
|------|------|------|
| 手机号输入框 | TextInput | 前缀+86，11位数字 |
| 验证码输入框 | TextInput | 6位数字 |
| 获取验证码按钮 | Button | 点击→倒计时60s→不可用 |
| 登录按钮 | Button | 校验通过→调登录 API→存 token→跳首页 |
| 注册账号链接 | Link | 跳转注册页面 |
| 密码输入框（注册） | TextInput | 6-20位，含字母+数字 |
| 昵称输入框（注册） | TextInput | 2-20字符 |

表单校验规则：
| 字段 | 校验规则 | 错误提示 |
|------|---------|---------|
| 手机号 | 必填，11位数字，以1开头 | "请输入正确的手机号" |
| 验证码 | 必填，6位数字 | "请输入6位验证码" |
| 密码（注册） | 必填，6-20位，含字母+数字 | "密码需6-20位，包含字母和数字" |
| 昵称（注册） | 必填，2-20字符 | "昵称需2-20个字符" |

导航关系：登录页→[注册链接]→注册页→[注册成功]→首页 | 登录页→[登录成功]→首页

#### 忘记密码页

交互元素表：
| 元素 | 类型 | 说明 |
|------|------|------|
| 手机号输入框 | TextInput | 前缀+86，11位数字 |
| 验证码输入框 | TextInput | 6位数字，发送验证码后启用 |
| 获取验证码按钮 | Button | 点击→倒计时60s |
| 新密码输入框 | TextInput | 6-20位，含字母+数字 |
| 确认新密码输入框 | TextInput | 与新密码一致校验 |
| 重置密码按钮 | Button | 校验通过→调 POST /auth/reset-password → 跳登录页 |

表单校验规则：
| 字段 | 校验规则 | 错误提示 |
|------|---------|---------|
| 手机号 | 必填，11位数字，以1开头 | "请输入正确的手机号" |
| 验证码 | 必填，6位数字 | "请输入6位验证码" |
| 新密码 | 必填，6-20位，含字母+数字 | "密码需6-20位，包含字母和数字" |
| 确认密码 | 必填，与新密码一致 | "两次输入的密码不一致" |

导航关系：登录页→[忘记密码]→忘记密码页→[重置成功]→登录页
