## 最终结果（来自结果蓝图）

### 要实现的内容

**单元测试（pytest）** — 覆盖核心逻辑：
- 体质评分算法（已知输入→预期输出）
- 组合引擎（排序/分配/去重/性平衡/兜底）
- 替换逻辑（同条件检索/去重检查）
- 权重更新（好评/差评→权重变化）
- 意图理解 LLM 输出解析

**集成测试（pytest + httpx）** — 覆盖所有 API 端点：
- 注册→登录→鉴权
- 评估→方案生成→查询→替换→反馈
- 内容查询
- 异常路径（无 token/过期/不存在/参数错误）
