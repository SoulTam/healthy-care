## 最终结果（来自结果蓝图）

### 要实现的内容

**语义检索模块**：
- Chroma similarity_search（向量相似度）
- 结合 metadata filter 缩小搜索范围
- 返回 top_k 候选（带相似度分数）

**组合引擎**：
```
排序优先级：体质完全匹配 > 季节匹配 > 症状匹配 > 功效匹配
餐类分配规则：
  - 早餐 → 粥/糊类
  - 午餐 → 主食 + 热菜 + 汤
  - 晚餐 → 轻食/汤羹
食材去重：同一方案内避免食材过度重复
性味平衡：同一餐内避免全寒或全热
兜底策略：
  - 无完全匹配 → 放宽症状条件
  - 仍无 → 同体质通用食谱随机推荐
```
