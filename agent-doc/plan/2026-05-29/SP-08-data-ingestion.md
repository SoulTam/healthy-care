## 最终结果（来自结果蓝图）

### 要实现的内容

**API**：
| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/retrieval/ingest-recipe` | POST | 入库单条食谱 |
| `/api/v1/retrieval/ingest-knowledge` | POST | 入库知识切片 |
| `/api/v1/retrieval/ingest/batch` | POST | 批量入库 |
| `/api/v1/retrieval/stats` | GET | 索引统计 |

**入库流程**：
```
食谱JSON → 解析 → LLM辅助打标 → embedding向量化 → 存入Chroma(向量+metadata)
                                                       → 存入SQLite(业务数据)
```

**初始食谱数据**：至少 50 道食谱，覆盖九种体质 + 四季，包含：名称、类别、体质适应、季节适应、食材、功效、性味归经、症状关联、禁忌、烹饪步骤、营养成分、古籍出处

**食谱入库格式**：
```yaml
食谱名称: "百合莲子银耳羹"
类别: 甜品/汤羹
体质适应: ["阴虚质", "气虚质"]
季节适应: ["秋季", "冬季"]
食材: ["百合", "莲子", "银耳", "冰糖"]
功效: ["滋阴润肺", "清心安神", "养胃生津"]
性味归经: {百合: "甘微寒，归心肺经", ...}
症状关联: ["干咳", "失眠", "口干", "便秘"]
禁忌: ["风寒咳嗽", "脾胃虚寒泄泻"]
烹饪步骤: "1.银耳泡发撕小朵 2.百合莲子洗净 3.加水炖煮1小时 4.加冰糖调味"
营养成分: {热量: 120, 蛋白质: 3, 脂肪: 0.5, 碳水: 28}
古籍出处: "《随息居饮食谱》"
```
